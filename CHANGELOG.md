# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-04
### Added
- Initial stable release of GoProxy TUI.
- Interactive TUI for real-time HTTP/HTTPS interception.
- MITM capabilities with dynamic certificate generation.
- Global storage for certificates and config (AppData/ConfigDir).
- Branding update: Identity changed to "GoProxy".
- Features: Domain blocking, domain ignoring, and header interception.
